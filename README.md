# Iris Flower Classification 🌸🌿

## 📌 Overview
This project uses the **Iris dataset** to classify flowers into three species:  
- Iris-setosa  
- Iris-versicolor  
- Iris-virginica  

It is part of my **CodeAlpha Data Science Internship** (Registration ID: CA/DF1/138844).  
The dataset is a classic benchmark in machine learning and is widely used for classification tasks.
